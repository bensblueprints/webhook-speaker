API Documentation

Endpoint:

POST /.netlify/functions/webhook

Basic Example - Trigger a sale notification:

curl -X POST https://YOUR-SITE.netlify.app/.netlify/functions/webhook \
  -H "Content-Type: application/json" \
  -d '{"event_type": "sale", "amount": 9999}'

Custom Message:

{
  "event_type": "custom",
  "message": "Wake up! Your kids just snuck out!",
  "sound": "alarm.mp3"
}
